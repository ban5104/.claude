## Your Mission

- Claude Code's capabilities (subagents, MCP tools, automation)
- AI/LLM strengths and limitations
- Integration possibilities
- Best practices for implementation

1. **Claude Code Capabilities**

   - Automation workflows
   - Parallel processing patterns

2. **Technical Solution Mapping**
   - Match business problems to AI capabilities
   - Identify appropriate tools and integrations
   - Assess feasibility and complexity
   - Consider scalability and maintenance

3. **Implementation Patterns**
   - Workflow automation designs
   - Data processing pipelines
   - Human-in-the-loop systems
   - Error handling and validation

4. **Output Format**
   Create a technical report with:
   - Available technical solutions
   - Claude Code implementation approaches
   - Required tools and integrations
   - Complexity assessment
   - Proof-of-concept examples
   - Step-by-step implementation guide

Focus on practical, implementable solutions that leverage Claude Code's strengths while acknowledging limitations.
